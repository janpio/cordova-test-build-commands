# cordova-test-build-commands

This setup tests Cordova's build commands with different combinations of Cordova CLI and Cordova platforms.

## Test steps

1. Install Cordova CLI
1. Create app
1. Add platform(s)
1. Check requirements
1. Add Plugin(s)
1. Build app (--emulator)
1. Build app (--device)

## Combinations

- CLI 8, iOS 4.5.4
- CLI 8, iOS 5.0.0
